# Skyway

English | [中文](./README_CN.md)

**Skyway** is a powerful AI-powered desktop application that combines intelligent development assistance with versatile automation capabilities.

## Two Modes, Infinite Possibilities

### 🛠️ Development Mode

A full-featured AI coding assistant for developers:

- 💬 **Intelligent Chat** - Natural language conversations to solve coding problems
- 🤖 **Multi-Provider Support** - Compatible with Claude, OpenAI, and custom AI providers
- 📁 **Project Workspaces** - Organize multiple projects with isolated workspaces
- 🌳 **File Tree** - Visual file browser with syntax-highlighted previews
- 🔀 **Git Worktree Isolation** - Each Agent works in its own isolated worktree, keeping your main codebase safe from experimental changes
- 💻 **Integrated Terminal** - Multiple terminal sessions within the app
- 🖥️ **IDE Integration** - Seamless connection with VS Code and other IDEs

### ⚡ Automation Mode

A lightweight mode for quick AI interactions without project context:

- 🚀 **Quick Start** - Start conversations instantly without opening a project
- 💡 **General Purpose** - Ask questions, brainstorm ideas, draft content, and more
- 🎯 **Task Automation** - Automate repetitive tasks with AI assistance
- 📋 **No Setup Required** - Perfect for quick questions and everyday tasks

## Common Features

- 🔧 **Custom Providers** - Configure your own AI endpoints with OpenAI-compatible APIs
- 🧩 **MCP Support** - Model Context Protocol for extended AI capabilities
- ⚡ **Skills System** - Install and manage AI skills for specialized tasks
- 🔌 **Plugin Marketplace** - Discover and install community extensions
- 🌙 **Dark Mode** - Full dark theme support
- 🌐 **i18n** - Available in English and Chinese
- 🔒 **Secure Storage** - Tokens and credentials stored securely

## Installation

### macOS

Download the latest `.dmg` file from [Releases](https://github.com/weavefox/skyway/releases) and drag to Applications.

**System Requirements:** macOS 12.0 or later

## Quick Start

1. **Choose Your Mode**
   - **Development Mode** - Open a project folder for full coding assistance
   - **Automation Mode** - Start chatting immediately without a project

2. **Configure Provider** - Set up your preferred AI provider (Claude, OpenAI, or custom)

3. **Start Working** - Code, ask questions, or automate tasks with AI
